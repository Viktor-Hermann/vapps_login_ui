## ⚡ **Installation**

 **Requirements - Flutter SDK should be present + VSCode**
 
1. Clone Project.
2. Open Terminal in VSCode in root folder of project and run
   ```
   flutter pub get
    ```
3. Run app with by pressing F5 on VSCode.

4. Enjoy.
